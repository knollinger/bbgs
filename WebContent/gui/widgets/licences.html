"Wenn ich weiter sehen konnte, so deshalb, weil ich auf den Schultern von Riesen stand."
Sir Isaac Newton

Freundliche Riesen kommen heute aus der OpenSource-Gemeinschafft. Hocken wir uns bei denen auf die Schulter, schaugn a weng nach links und rechts...der Ausblick ist fantastisch! Der alte Isaac hätte seine helle Freude...

Ehre wem Ehre gebührt, ohne folgende Projekte dads auch das hier nicht geben:

MySQL als Datenbank-Motor GNU public licence
Tomcat als Servlet-Engine Apache licence
ASCII Doc(tor) zur generierung von PDFs MIT licence
Das Oxygen Icon Set (great!!!) GNU Lesser General Public License
CKEditor als RichText-Editor GPL, LGPL und MPL
Eclipse als Entwicklungs-Umgebung Eclipse public licence
Sourceforge als Projekt-Hoster https://sourceforge.net/projects/bbgs/


Einen großem Dank an die Contributoren der genannten Projekte, das alles ist fantastische Software!
